Script to download galleries from Nh\*\*\*\*i to a local directory.

Download scripts from the `Release` tab.

Windows : `nhsync.exe`

Linux : `nhsync`

Pass the arguments `out-dir` and `api-key` upon running the script.

`out-dir` should be the absolute path to the local directory.

Example: `.\Downloads\nhsync.exe --out-dir="C:\Users\anon\Downloads\SomeDirectory" --api-key="my-api-key"`

Logs will be created beside where the script is located, and will be truncated when the script starts.

If you encounter any problems, feel free to open an [issue](https://github.com/agw2005/nhsync/issues).
